# Bulk-Music-Downloader

This program will convert YouTube videos into their highest quality audio files.

How to use: The program will ask for a YouTube playlist url or an individual YouTube video url. Depending on which the user chooses,
the program will either download all audio files from a YouTube playlist or download the audio file of a single YouTube video. Once
the user has made their choice, the program will prompt the user for a save directory via a GUI. Once selected, download will begin.
Incorrect choices and accidents have been handled for in the code.
